## Run Locally

Clone the project

```bash
  git clone https://github.com/sumansauravmay/KFC-clone.git
```

Go to the project directory

```bash
  cd kfctwo
```

Install dependencies for frontend as well as backend

```bash
  cd kfctwo
  npm install
```

```bash
  cd backend
  npm install
```

Start the localhost server

```bash
  cd frontend
  npm start
```
